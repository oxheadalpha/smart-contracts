
## Building

To build the Lorentz FA1.2, a.k.a. `ManagedLedger` contract, run:

```bash
make fa12
```

The result will be output to `out/fa12_lorentz.tz`

## Running

You can run the `morley-ledgers` executable using:

```bash
make run ARG="[morley-ledgers arguments]"
```
